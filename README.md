# Simple Links

A simple and minimalistic link manager for your browser.

## Features

- Add, edit, and delete links
- Add, edit, and delete groups
- Search for links and groups
- Sort links and groups
- Filter links and groups by tags
- Import and export links and groups
- Save links and groups to localStorage
- Responsive design
- Dark mode
- Mobile friendly

## Installation

1. Clone this repository
2. Open index.html in your browser
3. Start adding links and groups
